TESTES COM USUÁRIO FINAL


É importante ressaltar que o projeto de audiogame deve, além de seguir boas recomendações de desenvolvimento, incluir durante todos os processos de elaboração do jogo, testes com jogadores com deficiência visual para que os usuários possam conceder feedbacks importantes, visando um jogo mais acessível e imersivo possível.

SOBRE O PROJETO

A pesquisa apresentada neste site é um projeto de Mestrado em Ciência da Computação na Linha de Pesquisa em Interação Humano-Computador, que vem ao encontro de iniciativas para o desenvolvimento de jogos para pessoas com deficiência visual, permitindo mais alternativas de inclusão na área de entretenimento digital.

Buscamos explorar o desenvolvimento de audiogames, que são jogos baseados principalmente em interface sonora, sendo jogos que podem ou não conter interface gráfica, já que podem permitir seu uso somente por meio da interface sonora.

As diretrizes apresentadas neste site levam em consideração estudos já existentes, critérios de usabilidade e de acessibilidade digital. O conjunto também disponibiliza pontos de verificação (PV) por meio de heurísticas de avaliação, que possibilite ao desenvolvedor realizar pequenas avaliações após a implementação de cada diretriz.

Esperamos que as diretrizes resultantes deste estudo, possam ser mais facilmente considerados no processo de desenvolvimento de jogos acessíveis para pessoas com deficiência visual, por meio de uma proposta bem definida de diretrizes criada a partir de recomendações mais difundidas na literatura, disponíveis para este objetivo.

D1. LINGUAGEM CLARA

Utilizar diálogos mais simples e claros possíveis, de modo que as instruções do jogo se tornem fáceis de serem compreendidas.
